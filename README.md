### 为什么做这样一个app

因为这个学期我们专业在学习安卓，而安卓我又刚好会一点，所以想挑战学习新的技术和知识，so，花了1个晚上学习了一下flutter的
基本语法，然后参考了这个[项目](https://github.com/shijiacheng/wanandroid_flutter)，因此在这个项目中代码写的特
别的奇怪（~~辣鸡~~），会慢慢的修改，万一重构了呢？
